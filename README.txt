The following repository contains code for data analysis in:

Islands of retroelements are major components of Drosophila centromeresChing-Ho Chang, Ankita Chavan, Jason Palladino, Xiaolu Wei, Nuno M. C. Martins, Bryce Santinello, Chin-Chi Chen, Jelena Erceg, Brian J. Beliveau, Chao-Ting Wu, Amanda M. Larracuente, and Barbara G. Mellone. 2019. BioRxiv doi: 10.1101/537357 

All scripts are organized by figure. Supplementary data files are found in doi:10.5061/dryad.rb1bt3j.

Please direct questions to Amanda Larracuente at alarracu@bio.rochester.edu.
